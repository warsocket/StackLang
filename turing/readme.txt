This map gives 3 examples which satisfies the 3 requirements of structured porgram theory:

[sequence.sos]: Executing one subprogram, and then another subprogram (sequence)
[branch.sos]: Executing one of two subprograms according to the value of a boolean expression (selection)
[while.sos]: Repeatedly executing a subprogram as long as a boolean expression is true (iteration)



See https://en.wikipedia.org/wiki/Structured_program_theorem for more information
